# surbtc-sellbot

## Usage

### Setup:

    git clone https://github.com/delta575/surbtc-sellbot.git
    cd ./surbtc-sellbot
    python3 -m venv ./env
    source ./env/bin/activate
    pip install -r requirements.txt

### Run:
#### Activate enviroment
    cd ./surbtc-sellbot
    source ./env/bin/activate
#### Run Sellbot
    python ./sellbot.py
